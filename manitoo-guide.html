마니또란 무엇인가요?

마니또는 비밀 친구 활동이에요. 내가 챙겨줄 친구(마니또 대상)가 정해지면, 상대방 모르게 몰래 작은 친절을 베푸는 활동이에요.

이 앱에서는 이모지로 활동을 기록하고, 포인트를 쌓으며 마니또 활동을 재미있게 진행할 수 있어요.

💡 마니또 공개 이벤트 날까지 내 마니또가 누구인지 비밀로 유지해요!

🌐 접속 방법

  • 스마트폰 또는 PC에서 크롬 브라우저를 열어요
  • 주소창에 https://class-manitoo.web.app 입력
  • 학생 / 교사 탭을 선택하고 로그인해요
👩‍🏫

교사 기능

교사 전용

🔐 1. 로그인

  • 접속 후 상단 교사 탭 선택
  • 교사 아이디와 비밀번호 입력 후 로그인
  • 교사 대시보드 화면으로 이동

👥 2. 학생 명단 등록

  • 학생 명단 탭 클릭
  • 이름과 학번(4자리) 입력 후 추가 버튼 클릭
  • 자동으로 생성된 6자리 임시 비밀번호를 학생에게 개별 전달
  • 전체 학생 등록 완료까지 반복
⚠️ 임시 비밀번호는 반드시 개별 전달! 학생들이 처음 로그인할 때 이 비밀번호가 필요해요. 목록에서 언제든지 확인 가능해요.
💡 비밀번호를 분실한 학생은 비번 초기화 버튼으로 새 임시 비밀번호를 발급해주세요.

🎲 3. 마니또 랜덤 배정

  • 마니또 배정 탭 클릭
  • 지금 랜덤 배정하기 버튼 클릭
  • 자동으로 1:1 랜덤 배정 (자기 자신은 절대 배정 안 됨)
  • 배정 결과 확인 (이름 + 학번 표시)
⚠️ 배정 후 학생들 화면에 마니또 대상이 즉시 표시돼요. 배정 전에 모든 학생이 로그인을 완료했는지 확인하세요.

⭐ 4. 미션 관리

  • 미션 관리 탭 클릭
  • 미션 내용 입력 후 등록 버튼 클릭
  • 등록된 미션은 학생 화면 하단 미션 목록에 자동 표시

미션 예시: "마니또에게 응원 메모 몰래 남기기", "마니또가 힘들 때 말 한마디 건네기"

💡 학생들이 미션을 수행할 때 활동 이모지에서 ⭐ 미션 수행 이모지를 선택해서 기록해요.

🏆 5. 포인트 현황 & CSV 다운로드

  • 포인트 현황 탭 클릭
  • 학생별 포인트 순위 실시간 확인
  • CSV 다운로드 버튼으로 엑셀 파일 저장
💡 🤷 반응이 많은 학생은 마니또 활동이 잘 전달되지 않는 경우예요. 조용히 해당 학생을 독려해주세요.

🎊 6. 마니또 공개 이벤트

  • 공개 이벤트 탭 클릭
  • 공개할 학생 옆 공개하기 버튼 클릭
  • 해당 학생 화면에만 마니또 이름(학번)이 공개됨
  • 한 명씩 순서대로 공개하며 이벤트 진행
⚠️ 공개는 한 명씩 진행돼요. 한 번 공개하면 되돌릴 수 없어요!
💡 공개 전에 포인트 현황 → CSV 다운로드로 최종 결과를 저장해두세요.

🔄 7. 리셋 (새 마니또 시작)

  • 설정 탭 클릭
  • 전체 초기화(리셋) 버튼 클릭
  • 경고창 확인 후 포인트 저장 여부 선택
  • 초기화 완료 → 새 마니또 운영 시작 가능
⚠️ 리셋하면 모든 데이터가 영구 삭제돼요! 반드시 CSV를 먼저 저장하세요.
🧑‍🎓

학생 기능

학생 전용

🔐 1. 첫 로그인 & 비밀번호 변경

  • 접속 후 학생 탭 선택
  • 학번(4자리)과 선생님께 받은 임시 비밀번호(6자리) 입력
  • 나만 아는 새 비밀번호(4자리 숫자) 설정
  • 비밀번호 확인 입력 후 설정 완료 클릭
  • 학생 대시보드로 이동 + 로그인 보너스 +3점 적립!
⚠️ 새 비밀번호는 친구들이 유추하기 어려운 숫자로 설정하세요. 비밀번호를 잊으면 선생님께 초기화를 요청하세요.

🎯 2. 내 마니또 대상 확인

대시보드 상단에 내가 챙겨줄 친구 이름(학번)이 표시돼요.

선생님이 배정하기 전까지는 대기 화면이 표시돼요.

⚠️ 내 마니또 대상은 절대 다른 친구에게 알려주지 마세요! 공개 이벤트 날까지 비밀이에요.

📤 3. 활동 이모지 보내기

마니또 대상에게 무언가를 해줬을 때 이모지로 기록해요.

🎁선물/간식
💬응원/말걸기
🤝도움
😄웃게 해줌
📝메모/편지
⭐미션 수행
  • 이모지 6개 중 해당하는 것 선택 (파란 테두리로 표시)
  • 이모지 보내기 (+10점) 버튼 클릭
  • +10점 적립 완료!
⚠️ 하루 최대 2회만 보낼 수 있어요. 오늘의 2회를 다 쓰면 내일 다시 보낼 수 있어요.
💡 이모지는 매일 아침 7시에 상대방 피드에 공개돼요. 신중하게 보내세요!

📥 4. 받은 피드 & 반응 이모지

매일 아침 7시, 내 마니또가 보내준 활동 이모지가 피드에 공개돼요.

  • 받은 활동 피드 카드에서 이모지 확인
  • 해당 활동에 대한 반응 이모지 선택
  • 반응 이모지 전송 → 나에게 +10점, 내 마니또에게 보너스 +5점!
🥰너무 고마워요
😊기분 좋아요
😲깜짝 놀랐어요
🤗힘났어요
💛따뜻했어요
🤷모르겠어요
⚠️ 반응 이모지는 활동 1개당 1회만 가능하고 수정은 1번만 할 수 있어요.
💡 🤷를 선택하면 포인트는 없지만 솔직한 반응이에요. 선생님이 파악해서 마니또 활동을 도와줄 거예요.

🏆 5. 포인트 & 미션 확인

대시보드 상단 보라색 배너에서 내 총 포인트를 실시간으로 확인할 수 있어요.

화면 하단 미션 목록에서 선생님이 등록한 미션을 확인하고 수행해보세요!

💡 미션을 수행했다면 활동 이모지에서 ⭐ 미션 수행을 선택해서 보내세요.
💰

포인트 시스템

활동 포인트 비고
매일 로그인 +3점 하루 1회만 적립
활동 이모지 전송 +10점 하루 최대 2회
반응 이모지 전송 (긍정) +10점 받은 사람에게 적립
반응 이모지 보너스 +5점 활동을 보낸 마니또에게 적립
반응 이모지 (🤷) +0점 양쪽 모두 0점
💡 미션 1회 완수 시 총 포인트: 보내는 사람 +10 + 받는 사람 +10 + 보너스 +5 = 최대 25점!
📌

주의사항 & 자주 묻는 질문

❓ 비밀번호를 잊어버렸어요

선생님께 비밀번호 초기화를 요청하세요. 선생님이 새 임시 비밀번호를 알려드릴 거예요.

❓ 오늘 이미 이모지를 2번 보냈어요

하루 최대 2회 제한이에요. 내일 아침이 되면 다시 2회가 충전돼요!

❓ 받은 피드에 아무것도 안 보여요

활동 이모지는 매일 아침 7시에 공개돼요. 전날 받은 이모지가 오늘 아침 7시에 표시돼요.

❓ 마니또가 누구인지 알 것 같아요

공개 이벤트 날까지 절대 말하지 마세요! 비밀을 지키는 것도 마니또 활동의 일부예요 😊

❓ 이모지를 잘못 보냈어요

활동 이모지는 한 번 보내면 취소가 안 돼요. 신중하게 선택해주세요!

반응 이모지는 수정을 1번 할 수 있어요.
